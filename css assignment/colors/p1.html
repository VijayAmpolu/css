DEFINING COLORS VALUES

colors is css most often specifies in the following formates:

  1. COLOR KEYWORD-like red,green,blue,orange and etc..
  2. HEXADECIMAL 2^8=256 value - LIKE #0000, #ffff and etc..
  3. RGB (RED,GREEN,BLUE) value -like rgb(255,0,0) rgb(255,255,255) rgb(0,0,0,)
  4. RGBA (RED,GREEN,BLUE),ALPHA -like rgb(255,0,0,0.5)
  5. HSL(HUE,SATURATION,LIGHTNESS) value
  6. HSLA(HUE,SATURATION,LIGHTNESS,ALPHA) value
