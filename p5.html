CSS SELECTOR

we have some css selector to style the html element

  • Universal selector (*)
  • Element selector (tag name)(*)
  • Class name selector (class name using .)
  • Id selector (using id attribute)
  • Attribute selector (using any attribute inside the element)
  • Attribute with the value selector (attribute with value)
